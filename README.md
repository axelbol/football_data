# Real Data for Football Analysis

Only CSV files scraped from different sites.

If you want to get data from **FBREF**, **FOTMOB**, **SOFASCORE**, **UNDERSTAT** check this repository [scraping football websites](https://github.com/axelbol/scraping-football-sites).

## Repository details

__What can you get from libertadores25.csv?__

This CSV file contains detailed shot data from all matches in the 2025 Copa Libertadores tournament.

- **Comprehensive shot data** for every match in the tournament
- **Match metadata** including home/away team information
- **Round information** for easy filtering by tournament stage
- **Detailed shot characteristics** including:
  - Player name
  - Team name
  - Shot coordinates (x, y)
  - Minute of play
  - Blocked shot coordinates (blocked X, blocked Y)
  - More...
